# codetalk app / ReactNative
### Features
- Can create an account and log in. 
- A chat room can be created or entered into an already existing room. 
- In chat rooms, possible to message with other real people.
- Your login information will be stored as long as do not log out.
___________________
### Tools
- Formik
- Date-fns
- React-native-modal
- React-native-flash-message
- React Navigation
- Firebase - database/auth
- Vector Icons
